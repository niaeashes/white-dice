ダイスロール
d100 => 100面ダイスを1個振る
2d100 => 100面ダイスを2個振って合計を出す
2rd100 => 100面ダイスを1個振ることを2回繰り返す
d4+d6 => 4面ダイスの出目と6面ダイスの出目を合計する
s75 => 75% の技能ロール "s75 目星" みたいにコメントを付けても動作する

メッセージ・ダイスの他に、コマンドを入力することで、いろいろな追加機能を利用することができます。
/title (任意の文字列) ... タイトルを配信します。リロードするとリセットされます。
/character (キャラクター名) ... ステータスロールを行ないます。ログにも残ります。
/fortune ... おみくじを引きます。
/secret ... ダイスの出目を隠します。リロードすると見えなくなります。ログでは正常に見れます。
/select (項目) (項目) ... (項目) ... スペースで区切られた項目からダイスを振って一つを選びます。
/name (名前) (発言内容)... その発言のみ名前を変更します。
/help ... このメッセージを表示します。
※ このメッセージはログに残りません。
